"config.json"は設定ファイルです。
"config.json"をテキストエディタで開き、
{
    "language": "english"の部分を"japanese"に書き換えると、アプリケーションが日本語表示になります。
}
また、
{
    "generating_location": ""の部分にフォルダの場所を入力することで、テンプレートを生成する場所をそのフォルダに設定できます。
}
テンプレートは"addon_templete"フォルダに含まれる"templateBP/RP"フォルダとその中身を複製し、
その中にmanifestを自動生成する、という形で生成されるので、
このフォルダの中身を改変することで生成されるテンプレートの中身をカスタマイズすることができます。
